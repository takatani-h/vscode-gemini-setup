# CLAUDE.md

このファイルは、Claude Code（claude.ai/code）がこのリポジトリで作業する際のガイダンスを提供する。

## プロジェクト概要

開発ツールのセットアップ・インストールガイドをまとめた日本語ドキュメントリポジトリ。

- `gemini_cli.md` — Gemini CLI の導入・使い方ガイド
- `vscode.md` — Visual Studio Code の導入・初期設定ガイド

## 規約

- すべてのドキュメントは日本語で記述する。
- Markdown は GitHub Flavored Markdown に準拠する。
- 文体は「する」体（常体）を基本とする。
